# 子网存活主机探测 - 测试用例

## 编译

```powershell
cd SimplePortScanner/build
cmake ..
cmake --build . --config Release
```

## 测试 1：命令行 - 子网识别

```powershell
.\Release\SimplePortScanner.exe
# 选 4. 子网主机发现
```

**期望：**

- 列出本机网卡名称、本机 IP、CIDR（如 `192.168.1.0/24`）
- 扫描范围为 `192.168.1.1-192.168.1.254`

## 测试 2：命令行 - 存活探测

在菜单 4 中继续配置：

- 超时：300~500 ms
- 线程：20~50

**期望：**

- 本机 IP 通常显示为 `[ALIVE]`
- 网关（如 `192.168.1.1`）通常显示为 `[ALIVE]`
- 摘要显示 `存活主机：N / 254`

## 测试 3：衔接端口扫描

探测完成后选择 `是否使用存活 IP 继续端口扫描？(y/n): y`

**期望：**

- 自动带入存活 IP 列表，无需手动输入 IP
- 仅需输入端口表达式即可完成后续扫描

## 测试 4：Web 子网探测

```powershell
.\Release\SimplePortScanner.exe --web
```

浏览器打开 http://127.0.0.1:8080

1. 进入 **DISCOVER** 页
2. 选择子网，点击「开始探测」
3. 查看存活主机列表
4. 点击「用存活主机扫描」→ 跳转 SCAN 页，IP 已填入

## 测试 5：逗号分隔 IP 扫描

Web 或 CLI 扫描时，IP 表达式支持：

```
192.168.1.1,192.168.1.10,192.168.1.100
```

用于探测后离散存活主机的端口扫描。

## 答辩说明

| 概念 | 说明 |
|------|------|
| 主机发现 | 找哪些 IP 在线 |
| 端口扫描 | 找某 IP 哪些端口开放 |
| TCP 存活 | Open 或 Closed（RST）均表示有响应 |
| 子网来源 | `GetAdaptersAddresses` 读取本机网卡 IP/掩码 |
| 局限 | 防火墙屏蔽探测端口时可能漏报；仅支持 /24 |

## 安全提示

仅在本机、虚拟机、家庭路由器局域网或已授权实验环境中测试。
